Курсовая работа. Приложение реализующее работу с базой данных SQL Server. В качестве демонстрации была выбрана тема "Управление гостиницей". Доступны следующие функции:

* Добавление, редактирование, удаление записей в таблицах
* Поиск информации о сессиях(заселениях) по ID, паспорту
* Обработка неверного ввода записей пользователем

В качестве языка программирования был выбран C#, графический интерфейс WPF. Основная логика реализована в Tables.cs (класс Tables и унаследованные от него)
